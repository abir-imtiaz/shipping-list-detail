## Requirement

1. npm installed.
2. vue.js installed.

## Running

1. get this project
2. run " npm update "
3. install json-server by running " npm install -g json-server ".
4. start the project by running " npm run serve ".
5. open the fake json api by running " json-server --watch db.json ".
6. go to " localhost:8080 " on your browser.
